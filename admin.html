One-Time Setup (5 minutes)

  1. Add GA4 tracking to your site — Go to analytics.google.com → Admin → Create Property → Web. Copy your Measurement ID (G-B2QPQKD8MP) and paste it in the GA4 Measurement ID field below (for adding to page code).
  2. Get your Property ID — In GA4 → Admin → Property Settings. Copy the numeric Property ID (not the G-... one). Enter it in the banner above.
  3. Create an OAuth Client ID — Go to Google Cloud Console → Enable "Google Analytics Data API" → Credentials → Create OAuth 2.0 Client ID → Web Application → Authorised JS origins: https://www.shivacontinental.in. Enter the client ID in the banner above.
  4. Click Connect above — sign in with the Google account that owns the GA4 property. Data loads immediately.
