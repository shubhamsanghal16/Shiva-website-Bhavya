One step left: connect the database

The tracking code is live on every page and the API is deployed, but there is no database connected yet, so visits cannot be stored. Connect the free Redis database once and data starts flowing immediately:

  1. Open your project on vercel.com → Storage tab.
  2. Choose Upstash for Redis (free plan) → Create & Connect.
  3. Vercel adds UPSTASH_REDIS_REST_URL and UPSTASH_REDIS_REST_TOKEN automatically.
  4. Also add an ADMIN_KEY environment variable (your dashboard password).
  5. Redeploy the site. Done — this page turns into the live dashboard.
